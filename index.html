🛡️ Moderation Commands

/kick
Remove a member from the server. They can rejoin with a new invite link.
/kick [user] [reason]
⏱️ Cooldown: None 🔒 Permission: Kick Members
/ban
Permanently ban a member from the server. Prevents them from rejoining.
/ban [user] [reason]
⏱️ Cooldown: None 🔒 Permission: Ban Members
/mute
Timeout a member for a specified duration. They cannot send messages during timeout.
/mute [user] [duration] [reason]
⏱️ Cooldown: None 🔒 Permission: Moderate Members
/unmute
Remove timeout from a muted member, restoring their ability to send messages.
/unmute [user]
⏱️ Cooldown: None 🔒 Permission: Moderate Members
/clear
Bulk delete messages in the current channel. Maximum 100 messages at once.
/clear [amount]
⏱️ Cooldown: None 🔒 Permission: Manage Messages
/warn
Issue a formal warning to a member. Warnings are logged and tracked permanently.
/warn [user] [reason]
⏱️ Cooldown: None 🔒 Permission: Moderate Members
/warnings
View all warning records for a specific member including dates and reasons.

Discord.js v14

Built with the latest Discord.js library featuring full support for slash commands, buttons, select menus, and modern Discord features.

JSON Database

Lightweight local JSON-based database with automatic backup system. Supports user data, economy stats, and server configurations.

Permission System

Role-based permission checks ensure only authorized users can access moderation commands. Granular control over bot features.

Cooldown Manager

Smart cooldown system prevents command spam and abuse. Per-user tracking with automatic cleanup to optimize performance.

Error Handling

Comprehensive error handling and logging system. Graceful failure recovery ensures bot stability and reliability.

Analytics Tracking

Built-in analytics track command usage, user activity, and economy statistics for insights and optimization.
